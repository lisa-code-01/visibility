📦 一、变量的定义

JavaScript中有三种声明变量的方式：var、let 和 const

// var - 函数作用域，可重复声明 var name = "张三"; var name = "李四"; // 可以重复声明 // let - 块级作用域，不可重复声明 let age = 20; // let age = 25; // 报错：不能重复声明 age = 25; // 可以重新赋值 // const - 常量，声明时必须初始化，不可重新赋值 const PI = 3.14159; // PI = 3.14; // 报错：不能重新赋值

🔍 二、变量类型

JavaScript是动态类型语言，主要有以下数据类型：

基本数据类型

let str = "Hello"; // String 字符串 let num = 42; // Number 数字 let bool = true; // Boolean 布尔值 let empty = null; // Null 空值 let notDefined; // Undefined 未定义 let sym = Symbol('id'); // Symbol 符号 let big = 9007199254740991n; // BigInt 大整数

引用数据类型

let arr = [1, 2, 3]; // Array 数组 let obj = {name: "张三"}; // Object 对象 let fn = function() {}; // Function 函数 let date = new Date(); // Date 日期 let regex = /abc/; // RegExp 正则表达式

⚡ 五、函数

函数声明方式

// 1. 函数声明 function add(a, b) { return a + b; } // 2. 函数表达式 const multiply = function(a, b) { return a * b; }; // 3. 箭头函数 const subtract = (a, b) => a - b; // 4. 立即执行函数 (IIFE) (function() { console.log("立即执行！"); })();

函数参数

// 默认参数 function greet(name = "访客") { return `你好，${name}！`; } // 剩余参数 function sum(...numbers) { return numbers.reduce((a, b) => a + b, 0); } // 解构参数 function printUser({name, age}) { console.log(`${name}今年${age}岁`); }
