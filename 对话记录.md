## 学习知识点总结

### 变量基础
- **变量定义**：变量是存储数据的容器，使用 `变量名 = 值` 的语法创建
- **变量类型**：Python会自动判断变量类型，常见类型包括：
  - 整数（int）：如 18
  - 浮点数（float）：如 1.75
  - 字符串（str）：如 "Hello"
  - 布尔值（bool）：如 True 或 False

### 变量操作
- **重新赋值**：变量的值可以随时修改，后赋值会覆盖之前的值
- **变量运算**：变量可以直接参与数学运算（+、-、*、/、//、%）
- **变量交换**：Python支持简洁的变量交换方式：`x, y = y, x`
- **字符串操作**：字符串变量有许多实用方法（如 `len()`、`upper()`、`lower()`）

### 类型检查
- **type()函数**：用于返回变量的数据类型，用法：`type(变量名)`

## 练习题及解答

### 题目1：基础变量创建与输出
**要求**：创建三个变量，分别存储你的姓名、年龄和所在城市，然后输出"我叫[姓名]，今年[年龄]岁，来自[城市]"

**解答**：
```python
# 创建变量
name = "你的名字"
age = 18
city = "你的城市"

# 输出结果
print("我叫", name, "，今年", age, "岁，来自", city)
```

### 题目2：变量类型识别
**要求**：创建不同类型的变量，然后使用 `type()` 函数输出它们的类型

**解答**：
```python
# 创建变量
class_size = 45  # 整数
weight = 60.5    # 浮点数
course = "Python编程"  # 字符串
like_python = True  # 布尔值

# 输出类型
print("班级人数类型：", type(class_size))
print("体重类型：", type(weight))
print("课程名称类型：", type(course))
print("是否喜欢Python类型：", type(like_python))
```

### 题目3：变量重新赋值与类型转换
**要求**：创建一个变量，多次重新赋值不同类型的值，并输出每次的类型

**解答**：
```python
# 初始赋值
value = 100
print("初始值：", value, "类型：", type(value))

# 重新赋值为浮点数
value = 100.5
print("浮点数：", value, "类型：", type(value))

# 重新赋值为字符串
value = "100"
print("字符串：", value, "类型：", type(value))
```

### 题目4：变量运算应用
**要求**：创建变量存储圆的半径和π值，计算并输出圆的周长和面积

**解答**：
```python
# 存储半径和π
radius = 5
pi = 3.14

# 计算周长和面积
circumference = 2 * pi * radius
area = pi * radius ** 2  # ** 表示乘方

# 输出结果
print("圆的半径：", radius)
print("圆的周长：", circumference)
print("圆的面积：", area)
```

### 题目5：字符串变量操作
**要求**：创建一个字符串变量，输出其长度、大写形式、小写形式，并检查是否包含特定内容

**解答**：
```python
# 创建字符串变量
greeting = "Hello, Python!"

# 输出长度
print("字符串长度：", len(greeting))

# 输出大写
print("大写形式：", greeting.upper())

# 输出小写
print("小写形式：", greeting.lower())

# 检查是否包含"Python"
print("是否包含'Python'：", "Python" in greeting)
```
